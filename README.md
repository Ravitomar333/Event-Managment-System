# Event Management System

A comprehensive web-based platform for organizing events, managing vendors, and coordinating services. This Single Page Application (SPA) allows users to browse vendors, manage guest lists, and track orders, while providing vendors and admins with powerful management dashboards.

## Features

### üë§ User Portal
- **Browse Vendors**: Filter vendors by category (Catering, Florist, Decoration, Lighting).
- **Product Catalog**: View detailed product listings for each vendor.
- **Cart & Checkout**: Add items to cart, manage quantities, and place orders.
- **Guest List Manager**: Add and manage event guests.
- **Order Tracking**: Monitor the status of placed orders.
- **Request Items**: Submit custom item requests to vendors.

### üè™ Vendor Portal
- **Product Management**: Add, view, and manage products.
- **Order Processing**: View incoming orders and update their status (Received, Shipped, Delivered).
- **Transaction History**: Track sales and earnings.
- **Request Management**: Approve or reject custom item requests from users.

- **Vendor Management**: Onboard and manage vendors.
- **Order Overview**: View all orders across the platform.
- **Product Oversight**: Manage all products listed on the platform.
- **Membership**: Manage vendor memberships.

## Technologies Used
- **Frontend**: HTML5, CSS3 (CSS Variables, Flexbox, Grid)
- **Logic**: Vanilla JavaScript (ES6+)
- **Data Storage**: LocalStorage (No backend required)
